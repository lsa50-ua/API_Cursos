Para iniciar la aplicacion:
	- En ambas carpetas hay que ejecutar "npm i " para instalar las dependencias
	- Para arrancar el backend, desde la carpeta server "node server.js"
	- Para arrancar el frontend, desde la carpeta client ejecutar "npm run dev"
